/**
* 因为我是一个程序员，不太会表达自己的想法
* 所以在这里我想用小小程序来表达我对你的心意。
*/
Boy i = new Boy("安笨笨");
Girl u = new Girl("瑶宝宝");
// 2009年3月15的那一天傍晚, 我告诉你说我喜欢你。
i.love(u);
// 非常幸运, 你接受了我，答应做我的女朋友。
u.accepted(i);
// 从那一天起, 我每天都会想你，深深地思念。
i.miss(u);
// 并且我努力学会照顾你和我们的爱。
i.takeCareOf(u);
// 现在，你说你现在还不能嫁给我。
// 我并没有说太多，我会等待也会努力，因为我有信心娶你。
boolean isHesitate = true;
while (isHesitate) {
// 我同样也觉得这也是一个很重要的决定，所以我会等待
i.waitFor(u);
// 直到你想好了，答应嫁给我。
isHesitate = u.thinkOver();
}
// 一个浪漫的婚礼之后，我们一定会很幸福的在一起。
i.marry(u);
i.liveHappilyWith(u);
